# Ajjdxns-Minecraft-Launcher
> 一个使用Python写的全平台Minecraft启动器
### 安装方法
1.克隆本仓库
```
git clone https://github.com/ajjdxns-studio/Ajjdxns-Minecraft-Launcher.git
```
2.转到仓库目录
```
cd Ajjdxns-Minecraft-Launcher
```
3.转到主文件目录（PC）
```
cd PC
```
4.运行启动脚本，完工！
```
#windows
start.bat

#lunix
start.sh

#osX
python3 main.py
```
接下来，在命令行里输入`help`获取应用内帮助吧。
